# MilkGuide

> AI로 우유를 알려줄게!

### 소개글

우유 종류를 먹어보지 않으면 알기 어려운 시각장애인 분들을 위하여, 사진으로 손쉽게 우유 종류를 알려주는 우유 Detecting 앱입니다. 🌝

### 주요 기능

- 우유 찾기
  - 우유를 찍기 위한 카메라 기능
  - 우유 사진을 분석하는 ImageClassification 모델

- Setting
  - 버전, 앱 소개, 개발자 소개
  - 앱스토어 리뷰 남기러 가기
  - 뷰에 따른 오디오 안내 기능 on/off (추가예정)
